1 Critical — Daily-use gaps Reached for multiple times a day by every Finder user
Feature Finder Reference Status Notes
Batch Rename
 Rename Finder Items… Missing Multi-select files → rename together with format/replace/sequence rules. Single-file rename works; bulk rename has no UI.
Undo / Redo File Operations
 Edit → Undo Missing ⌘Z to undo last move/rename/trash/copy. No undo stack exists today — once trashed/moved, the user must navigate back manually.
Visible Drag & Drop Targets
 Drag & drop Partial Move semantics exist in IPC, but the FileList lacks drop-target highlighting, drag-into-folder affordance, and inter-pane drag handling.
Open With… Submenu
 Open With ▸ Missing Currently only "Open" (default app) and "Reveal in Finder". Missing list of alternative installed apps for the file's UTI.
Tabs Within Window
 ⌘T New Tab Missing Dual-pane exists, but no tab strip per pane. Users juggling many folders must cycle history or open new windows.
2 Important — Frequent power-user features Used weekly; significant workflow gaps without them
Feature Finder Reference Status Notes
Permissions Viewer / Editor
 Get Info → Sharing & Permissions Missing Get Info shows size/dates/path but not POSIX mode or owner/group. No UI to chmod/chown.
Copy / Move Progress UI
 Copy progress sheet Missing Cross-volume copies and large transfers happen synchronously with no progress bar, ETA, or cancel button.
File Comments / Notes
 Get Info → Comments Missing Free-text Spotlight comments per file. Tags exist; arbitrary notes do not.
Archive Formats Beyond Zip
 Archive Utility Partial Zip / unzip implemented. Missing tar, tar.gz, 7z extraction (read-only at minimum).
Sidebar Customization
 Finder → Settings → Sidebar Missing No way to reorder sections (Favorites / Smart / Recents / Volumes), hide them, or drag-reorder pinned folders.
Global / System-wide Search Scope
 This Mac vs This Folder Partial Search and smart folders are scoped to the current directory. No "search This Mac" toggle.
3 Advanced — Power-user / niche features Used occasionally; full parity with Finder requires these
Feature Finder Reference Status Notes
Spring-Loaded Folders
 Drag-hover open Deferred Hover a dragged item over a folder → folder opens after a delay so you can drill in mid-drag. Requires drop-target work first.
Symlink / Alias Creation
 Make Alias (⌘L) Deferred No UI to create symlinks. Existing symlinks are followed but not visually distinguished.
Multi-Window Support
 ⌘N New Finder Window Deferred Single Electron window only. Dual-pane partially compensates but doesn't replace independent windows.
iCloud Drive Surface
 Sidebar → iCloud Drive Deferred No native iCloud section, sync status pips, or "download / remove download" actions for cloud-only files.
Spotlight-Style Global Search
 Spotlight (⌘Space) Deferred In-folder content search is implemented; system-wide indexed search across the whole machine is not. Big project — would need an index.
